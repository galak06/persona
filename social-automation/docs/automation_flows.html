1. Discovery & Scoring

Scanners visit groups/hashtags. score_relevance filter posts. Qualifying posts are drafted inline via Gemini.

Data: comment_queue.json (status: pending)

2. Autonomous Approval

comment_approver.py runs periodically. It stamps items as approved if a draft exists, or skipped if the LLM failed.

3. Deployment

comment_poster.py picks up approved items and uses Playwright to perform the actual post on FB/IG.

Log: engagement_log.jsonl

📰 Blog & Campaign Pipeline

1. Ideate (Stage)

User runs /content-ideator. 5 ideas are generated and saved to enrichment_cache.json after Telegram confirmation.

2. Enrich (Stage)

System researches SEO/Context for the top idea. A detailed brief is sent to Telegram. Wait for "Approve".

3. Write & Publish (Stage)

Generates a WordPress draft. Once approved, pushes the link + custom captions to the Facebook Page and Instagram.

1. Group Discovery

Monthly cron searches for relevant groups based on niche keywords. Saves to pending_groups.json.

2. Management UI

User reviews groups in the Frontend "Groups" tab. Toggles "Join" or "Watch".

3. Execution

API triggers Playwright to visit the group and submit the join request (handling join questions if needed).
