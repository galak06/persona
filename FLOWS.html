3. Flows

3.1 Engagement loop — daily scan + comment

The legacy outbound engagement pipeline. Scans social platforms, scores posts, drafts comments in Nalla's Dad voice, gates on Telegram, posts.

site-analyzer
Crawls dogfoodandfun.com RSS + sitemap → data/site_content_cache.json. Comments reference current site content. stale: last run 2026-04-28
fb-scanner
Playwright-driven scan of joined dog groups. Scores by relevance (food/GPS/running/training). High-score posts (≥0.75) queued to comment_queue.json.
ig-scanner
Hashtag scan. Likes qualifying posts; queues top candidates for comments. Caps: 8 likes/d, 2 comments/d.
auto-drafter
Reads comment_queue.json, drafts Nalla's-Dad-voice replies, validates against blocked-terms list, writes recent_drafts.jsonl.
comment-approver
Hourly Telegram batch: surfaces drafts for approve/skip. Safe drafts can auto-approve.
comment-poster
Posts approved comments via Playwright (FB) or Graph API (IG). Logs to engagement_log.jsonl. last successful post 2026-05-06
reply-follower
Twice-daily revisits — scrapes new replies on posted comments, drafts conversational responses, Telegram-gates, posts threaded replies.

3.2 Content publishing — blog post → social

Cold-start content engine. Idea → SEO/social brief → WP draft → FB page → IG feed → FB groups.

content-ideator
Sunday weekly run. Reads content gaps, trends, PAA, seasonal hooks → 8-10 new ideas appended to Google Sheet posts tab.
content-enricher
For each publish-status idea: SEO research + competitor look-up + Nalla angle → enriched brief → Telegram approval gate.
wp-post-creator
Generates full 2,500-3,500-word post (Nalla's Dad voice, FAQ, comparison table, FTC disclosure, FIFU featured image) → WP draft via REST API.
fb-post-creator
After WP published: 150-200w FB page caption, no hashtags, posted via Graph API. Target 8-10am EST.
ig-post-creator
Same source → IG caption (80-150w, 6-8 hashtags, "link in bio"), Pexels image fetched, Graph API container→publish. Target 6-8pm EST, ≥4h after FB.
fb-group-publisher
≥3h after page post: shares permalink across eligible groups (rules-checked per group). Link inline in body — never "link in first comment" (Page-algo-only).

3.3 Recipe pipeline — seed-driven, two-phase

Recipe-specific engine. Operator gates on audio + final verify before the daily cron publishes. Detailed flow →

ideator
Weekly Sun research → 2-gate Telegram approval (idea, then enriched seed) → appended to seeds.json + seeds/carousels/{slug}.json.
prepare.py
Builds artifacts (recipe body, captions, 4 carousel slides, featured image, WP draft) into campaigns/prepared/<seed>/. Status: awaiting_audio.
operator audio
manual Drop audio.mp3 from Suno into the prepared folder. Status → audio_ready.
--verify
publish_prepared.py --seed <id> --verify sends Telegram preview (full body + slides + captions + audio). Approve → status verified.
publish (cron)
Daily 14:00 Israel — drains queue: WP draft→publish, IG carousel, FB page post, Pinterest pin Pinterest Trial-tier API. Idempotent on crash.

3.4 FB group growth — scout, warmup, distribute

fb-group-scout
Weekly Sun. Searches FB for new dog groups (public + private), scores against criteria → Telegram approve → join requests. Caps: 5/day, 15/week.
group warmup
untracked lib/group_warmup.py enforces 48h-comment / 72h-link-post cooldown on newly joined groups before publisher uses them.
fb-group-distribute
Daily 17:30 — picks eligible blog post + eligible groups, drafts category-tailored captions, posts. Cap: 10 link-shares/day (under FB's ~25/d ceiling).

3.5 Own-content tracking

fb-notification-scan
Weekly Sun 09:00. Pulls page notifications, queues incoming comments on our own posts for reply.
ig-own-comments
Hourly 09:30→. Watches incoming comments on our own IG posts. "Comment RECIPE" CTAs are fulfilled manually by the operator — never propose a DM bot.
wp-comment-handler
Daily 21:00. Pulls held comments via REST, auto-trashes spam (3+ URLs, spam keywords, spam TLDs), queues rest for Telegram approval, approve+reply in one shot. stale: last run 2026-04-22

3.6 IG follow-scout

untracked New flow: lib/ig_follow/ + scripts/ig_follow.py. Reads data/competitors.json (multi-consumer; same file feeds FB group search via fb_query), finds engagers + followers of competitor accounts, geo-filters to NA, follows at the daily cap (22/d, mid-band).
